# httpflooder
simple http flooder

## building
run:
```bash
go build
```

you agree that when using this tool, the maintainer will NOT be held responsible for any misuse!!! ⚠️

## example
```bash
./httpflooder -d 120 -i 1 -t 4 -c 8 -h 127.0.0.1 -p 8080
```
